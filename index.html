Abstract

Optical character recognition (OCR) on turbulence-degraded images is challenging due to spatially correlated blur and pixel-level distortions that disrupt character structures. An Autonomous Self-Supervised Fine-Tuning (ASFT) framework is proposed to improve OCR performance of Qwen2.5-VL-3B on book cover images affected by strong atmospheric turbulence (α = 0.7). A systematic evaluation of 14 adapter configurations on 5,000 turbulence-degraded samples identifies standard QLoRA (4-bit NF4, r = 32) as the most stable and effective quantized fine-tuning strategy.

A comprehensive architectural study across multiple self-supervision paradigms—including GRPO-based learning, consistency regularization, contrastive self-critique, and curriculum-based training—is conducted. The ASFT framework integrates a turbulence simulation pipeline, a hidden-state-driven quality evaluation mechanism, a lightweight self-critique module for representation refinement, and an adaptive curriculum strategy for dynamic difficulty control. On a held-out test set of 50 samples, ASFT achieves 93.27% text similarity, 88.18% word F1, 9.59% CER, and 14.57% WER, outperforming all evaluated supervision strategies and the zero-shot baseline by a significant margin.

Methodology

The ASFT framework is built on Qwen2.5-VL-3B fine-tuned with QLoRA (4-bit NF4, r = 32, α = 64). Three novel modules attach to final-layer hidden states during training only and are discarded at inference, adding zero computational overhead.

AQE Autonomous Quality Evaluator

Four scalar quality scores (accuracy, coherence, completeness, confidence) from mean-pooled hidden states via independent MLP heads (d→256→64→1, LayerNorm, GELU). A meta-decision network maps scores to a three-way decision: Accept (×0.9 loss), Refine (×1.0), or Reject (×1.5), focusing gradient on hard examples.

SCM Self-Critique Module

A single-layer Transformer encoder re-processes hidden states H to derive an improvement vector ΔH. Refined states H̃ = H + Wr[H; ΔH] are used in the consistency loss. Operates at training time only — zero inference overhead. Uses dmodel = 2048, 8 heads, pre-norm, dropout 0.1.

ACM Adaptive Curriculum Manager

Rolling window of size 50 over quality scores and acceptance rates, adapting the AQE quality threshold τt at each step. Increases difficulty by 0.05 when performance exceeds 0.90 and acceptance rate exceeds 0.80; decreases by 0.08 when performance falls below 0.70. Reached maximum difficulty after only 50 samples.

Composite Loss Function

ℒ = ℒCE + 0.35 · ℒqual − 0.25 · ℒcons
ℒqual = (1/4) ∑k (1 − sk)   // drives all 4 quality scores toward 1
ℒcons = −(1/T) ∑t sconf · cos(H̃t, Ht)   // consistency: refined vs original

Loss weights λ1 = 0.35 and λ2 = 0.25 selected by grid search over {0.10, 0.25, 0.35, 0.50} on validation set. Over-weighting quality loss (λ1 = 0.50) destabilises training by up to −3.73 pp.

Training Configuration

Effective batch size 16 (per-device 4, gradient accumulation 4), learning rate η = 2×10−4 with cosine schedule and 50 warm-up steps, BF16 precision, paged_adamw_8bit, gradient clipping 0.3. Dual Tesla T4 GPUs. Training time: ~40 min (400 samples) / ~368 min (full run).

Experiments

Dataset

500 book cover images (Project Gutenberg via Kaggle) degraded with our turbulence simulation pipeline: spatially correlated Gaussian displacement fields (Δx, Δy ∼ Ν(0, (α·4.5)2), α = 0.7), 11×11 Gaussian kernel smoothing, bilinear remapping, then final Gaussian blur (9×9, σb = 2.0). Split: 400 / 50 / 50 (train / val / test, seed 42).

Backbone Selection

Standard LoRA fine-tuning (r = 32, 2 epochs, 400 samples) applied to seven VLMs. OCR-specialist models (TrOCR, GOT-OCR) remain below 65% TS due to fixed-vocabulary decoders. General-purpose VLMs converge at 67–76% TS. Qwen2.5-VL-3B achieves 76.40% TS, outperforming LLaVA-1.6-7B by +25.74 pp despite fewer parameters — reasoning capability matters more than scale.

Adapter Method Selection (14 Configurations)

14 adapter strategies evaluated on 3,956 samples over 2 epochs on dual Tesla T4. Enhanced LoRA+ v1 diverged (loss 2.14) and LoRA+ Extreme collapsed (loss 4.04). Standard QLoRA achieves the lowest loss (0.5115) and is adopted for all ASFT experiments.

Results

ASFT achieves the best value on every metric across all 50 held-out test samples. Exact match reaches 38% (19/50) compared with 0% for the zero-shot baseline. Of 50 test samples, 37 (74%) fall in the Excellent tier (≥90% TS), 9 (18%) in Good (75–90%), and 4 (8%) in Fair (<75%). Per-sample standard deviation: ASFT 8.66% vs. base 11.35%.

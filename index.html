Abstract

Optical character recognition (OCR) on turbulence-degraded images is challenging due to spatially correlated blur and pixel-level distortions that disrupt character structures. An Autonomous Self-Supervised Fine-Tuning (ASFT) framework is proposed to improve OCR performance of Qwen2.5-VL-3B on book cover images affected by strong atmospheric turbulence (α = 0.7). A systematic evaluation of 14 adapter configurations on 5,000 turbulence-degraded samples identifies standard QLoRA (4-bit NF4, r = 32) as the most stable and effective quantized fine-tuning strategy.

A comprehensive architectural study across multiple self-supervision paradigms—including GRPO-based learning, consistency regularization, contrastive self-critique, and curriculum-based training—is conducted. The ASFT framework integrates a turbulence simulation pipeline, a hidden-state-driven quality evaluation mechanism, a lightweight self-critique module for representation refinement, and an adaptive curriculum strategy for dynamic difficulty control. On a held-out test set of 50 samples, ASFT achieves 93.27% text similarity, 88.18% word F1, 9.59% CER, and 14.57% WER, outperforming all evaluated supervision strategies and the zero-shot baseline by a significant margin.

Methodology

The ASFT framework is built on top of Qwen2.5-VL-3B fine-tuned with QLoRA (4-bit NF4, r = 32, α = 64). Three novel modules are attached to the final-layer hidden states during training only — they are discarded at inference, adding zero computational overhead to deployment.

AQE Autonomous Quality Evaluator

Four scalar quality scores (accuracy, coherence, completeness, confidence) are produced from mean-pooled hidden states via independent MLP heads (d→256→64→1, LayerNorm, GELU). A meta-decision network maps the quality vector to a three-way decision: Accept (×0.9 loss scale), Refine (×1.0), or Reject (×1.5), focusing gradient on hard examples.

SCM Self-Critique Module

A single-layer Transformer encoder re-processes hidden states H to derive an improvement vector ΔH. Refined hidden states H̃ = H + Wr[H; ΔH] are used in the consistency loss. The SCM operates only at training time, adding zero inference overhead. The encoder uses dmodel = 2048, 8 heads, pre-norm architecture with dropout 0.1.

ACM Adaptive Curriculum Manager

Maintains a rolling window of size 50 over quality scores and acceptance rates, adapting the AQE quality threshold τt at each update step. Difficulty increases by 0.05 when rolling-window performance exceeds 0.90 and acceptance rate exceeds 0.80; decreases by 0.08 when performance falls below 0.70. In experiments, the ACM reached maximum difficulty after only 50 rolling samples.

Composite Loss Function

ℒ = ℒCE + 0.35 · ℒqual − 0.25 · ℒcons

ℒqual = (1/4) Σk (1 − sk)    // drives all 4 quality scores toward 1

ℒcons = −(1/T) Σt sconf · cos(H̃t, Ht)    // consistency between refined and original

The loss weights λ1 = 0.35 and λ2 = 0.25 were selected by grid search over {0.10, 0.25, 0.35, 0.50} on the validation set. The optimal cell (0.35, 0.25) achieves 93.27% TS. Over-weighting the quality loss (λ1 = 0.50) destabilises training by up to −3.73 pp.

Training Configuration

Effective batch size 16 (per-device batch 4, gradient accumulation 4), learning rate η = 2×10−4 with cosine schedule and 50 warm-up steps, BF16 precision, paged_adamw_8bit, gradient clipping at 0.3. All experiments run on dual Tesla T4 GPUs. Training time: ~40 minutes for the 400-sample run and ~368 minutes for the full ASFT run.

Experiments

Dataset

500 book cover images from the Book Image Dataset (Project Gutenberg, via Kaggle) are degraded using a turbulence simulation pipeline. Each image is distorted by spatially correlated Gaussian displacement fields (Δx, Δy ~ 𝒩(0, (α·4.5)2), α = 0.7), smoothed with an 11×11 Gaussian kernel, followed by bilinear remapping and a final Gaussian blur (9×9, σb = 2.0). The 500 images are split 400 / 50 / 50 (train / val / test, seed 42).

Backbone Selection

Standard LoRA fine-tuning (r = 32, 2 epochs, 400 samples) is applied to seven VLMs. OCR-specialist models (TrOCR, GOT-OCR) remain below 65% TS due to fixed-vocabulary decoders. General-purpose VLMs converge in the 67–76% TS range. Qwen2.5-VL-3B achieves 76.40% TS, outperforming LLaVA-1.6-7B by +25.74 pp despite having fewer parameters, indicating that reasoning capability matters more than model scale for turbulence-degraded OCR.

Adapter Method Selection (14 Configurations)

14 adapter strategies are evaluated on Qwen2.5-VL-3B using 3,956 samples over 2 epochs on dual Tesla T4 GPUs. Full-precision LoRA variants (BF16) achieve slightly lower losses (0.5165–0.5181) but require 381–444 minutes. Advanced strategies including rsLoRA with LISA and QDoRA add overhead without improving loss. Enhanced LoRA+ v1 diverged (loss 2.14) and LoRA+ Extreme collapsed (loss 4.04). Standard QLoRA (run 1) achieves the lowest loss (0.5115) among all quantized methods and is adopted for all ASFT experiments.

Supervision Layer Architecture Search

Seven layer configurations are compared, all built on the same QLoRA backbone and trained on 400 samples for 2 epochs. Adding any single component (VSS, COS, PCL) improves over standard QLoRA but converges below 90% TS. Combining two components narrows the gap but still falls short by at least 4.3 pp. Only Full ASFT (L7) with all three components crosses 93% TS, confirming their complementary and non-redundant contributions.
